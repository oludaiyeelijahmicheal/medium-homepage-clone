Medium Homepage Clone

A responsive clone of the Medium homepage built using HTML and CSS.

Overview

This project recreates the homepage of Medium as a frontend practice project. It demonstrates my understanding of HTML5 and CSS3, including page structure, styling, and responsive layout techniques.
